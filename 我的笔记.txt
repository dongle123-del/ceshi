   Ubuntu 基本命令操作
一、系统管理

    查看系统信息
        uname -a：查看内核版本及系统信息
        lsb_release -a：查看 Ubuntu 版本信息
        hostname：查看主机名
        hostnamectl set-hostname 新主机名：修改主机名（需重启生效）
    用户管理
        adduser 用户名：创建新用户（交互式，会自动创建家目录）
        userdel -r 用户名：删除用户及家目录
        su - 用户名：切换用户（- 表示加载目标用户环境变量）
        sudo 命令：以管理员权限执行命令
        passwd 用户名：修改用户密码（当前用户可直接用passwd）
    软件包管理（apt）
        sudo apt update：更新软件源列表
        sudo apt upgrade：升级所有已安装软件
        sudo apt install 软件名：安装软件
        sudo apt remove 软件名：卸载软件（保留配置）
        sudo apt purge 软件名：彻底卸载软件（删除配置）
        sudo apt autoremove：自动清理无用依赖包
        apt search 关键词：搜索软件
    进程管理
        ps aux：查看所有进程（a：所有用户进程；u：显示用户；x：包括非终端进程）
        top：动态查看进程资源占用（按 q 退出）
        kill 进程ID：终止进程（默认 SIGTERM 信号）
        kill -9 进程ID：强制终止进程（SIGKILL 信号）
        pkill 进程名：按名称终止进程
    服务管理（systemd）
        sudo systemctl start 服务名：启动服务
        sudo systemctl stop 服务名：停止服务
        sudo systemctl restart 服务名：重启服务
        sudo systemctl enable 服务名：设置服务开机自启
        sudo systemctl disable 服务名：取消开机自启
        sudo systemctl status 服务名：查看服务状态

二、文件与目录操作

    目录导航
        cd 目录路径：切换目录（cd ~ 回到家目录；cd .. 回到上级目录；cd - 回到上一次目录）
        pwd：显示当前目录路径
    文件 / 目录查看
        ls：列出当前目录内容（ls -l 详细列表；ls -a 显示隐藏文件；ls -lh 人性化显示大小）
        tree 目录：以树状结构显示目录内容（需先安装tree包）
        cat 文件名：查看文件全部内容
        more 文件名：分页查看文件（按空格翻页，q 退出）
        less 文件名：分页查看文件（支持上下键滚动，q 退出）
        head -n 行数 文件名：查看文件前 N 行（默认 10 行）
        tail -n 行数 文件名：查看文件后 N 行（tail -f 文件名 实时追踪文件更新）
    文件 / 目录操作
        mkdir 目录名：创建目录（mkdir -p 父目录/子目录 递归创建多级目录）
        rmdir 目录名：删除空目录
        touch 文件名：创建空文件或更新文件时间戳
        cp 源路径 目标路径：复制文件 / 目录（cp -r 源目录 目标目录 复制目录及内容）
        mv 源路径 目标路径：移动 / 重命名文件 / 目录
        rm 文件名：删除文件（rm -r 目录 删除目录及内容；rm -f 文件名 强制删除，不提示）
    文件权限
        chmod 权限 文件名：修改文件权限（如chmod 755 file：所有者读写执行，组和其他读执行）
        chown 用户名:组名 文件名：修改文件所有者及所属组（chown -R 递归修改目录）

三、磁盘与网络

    磁盘管理
        df -h：查看磁盘分区使用情况（h：人性化显示）
        du -sh 目录：查看目录占用空间（s：汇总；h：人性化）
    网络管理
        ip addr：查看网络接口及 IP 地址
        ping 域名/IP：测试网络连通性（按 Ctrl+C 停止）
        netstat -tuln：查看监听端口（t：TCP；u：UDP；l：监听；n：数字显示）
        ss -tuln：类似 netstat，更高效
        curl 网址：发送 HTTP 请求并显示响应（如curl http://www.baidu.com）
        wget 下载链接：下载文件（wget -O 文件名 链接 指定保存文件名）

Docker 基本命令操作
一、Docker 服务管理

    sudo systemctl start docker：启动 Docker 服务
    sudo systemctl stop docker：停止 Docker 服务
    sudo systemctl restart docker：重启 Docker 服务
    sudo systemctl enable docker：设置 Docker 开机自启
    docker --version：查看 Docker 版本
    docker info：查看 Docker 详细信息

二、镜像（Image）操作

    镜像拉取与搜索
        docker pull 镜像名:标签：拉取镜像（标签默认 latest，如docker pull ubuntu:20.04）
        docker search 关键词：搜索 Docker Hub 上的镜像
    镜像管理
        docker images：列出本地所有镜像（docker images -a 包括中间层镜像）
        docker rmi 镜像ID/镜像名:标签：删除镜像（若有容器依赖需先删除容器，-f 强制删除）
        docker tag 原镜像名:标签 新镜像名:新标签：给镜像打标签
        docker save -o 保存路径.tar 镜像名:标签：导出镜像
        docker load -i 镜像文件.tar：导入镜像

三、容器（Container）操作

    容器创建与启动
        docker run [选项] 镜像名 [命令]：创建并启动容器
            常用选项：
                -it：交互式终端（如docker run -it ubuntu /bin/bash 进入 Ubuntu 容器交互模式）
                -d：后台运行容器（守护态，如docker run -d nginx）
                --name 容器名：指定容器名称（如docker run --name mynginx nginx）
                -p 主机端口:容器端口：端口映射（如docker run -p 8080:80 nginx，主机 8080 映射容器 80）
                -v 主机目录:容器目录：数据卷挂载（如docker run -v /host/data:/container/data nginx）
    容器状态管理
        docker ps：列出运行中的容器（docker ps -a 列出所有容器，包括停止的）
        docker start 容器ID/容器名：启动已停止的容器
        docker stop 容器ID/容器名：停止运行中的容器
        docker restart 容器ID/容器名：重启容器
        docker kill 容器ID/容器名：强制停止容器
    容器交互与信息
        docker exec -it 容器ID/容器名 命令：进入运行中的容器（如docker exec -it mynginx /bin/bash）
        docker logs 容器ID/容器名：查看容器日志（-f 实时追踪日志）
        docker inspect 容器ID/容器名：查看容器详细信息（包括 IP、挂载等）
    容器删除
        docker rm 容器ID/容器名：删除已停止的容器（-f 强制删除运行中的容器）
        docker rm $(docker ps -aq)：删除所有容器（谨慎使用）

四、其他常用命令

    docker volume ls：列出所有数据卷
    docker volume create 卷名：创建数据卷
    docker volume rm 卷名：删除数据卷
    docker network ls：列出所有网络
    docker system prune：清理未使用的镜像、容器、网络（-a 清理所有未被使用的资源）
